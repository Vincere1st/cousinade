# Cousinade

Site pour aider à l'organisation d'une cousinade.

## Techno
Vue3 et Vuetify3 en frontend

Supabase en backend

## Setup

Installation des dépendences:

```bash
# yarn
yarn install

# npm
npm install

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```
